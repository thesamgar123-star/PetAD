# Animal Shelter (OOP Assignment 2)

My implementation of a shelter management system in Java. The project covers basic OOP principles like inheritance and polymorphism, and uses collections for data storage.

## Features
- Animal and Pet hierarchy (inheritance)
- ArrayLists for storing data
- Search logic for finding records
- Custom equals/hashCode/toString methods
- Console interaction via Scanner

## Classes
- Animal & Pet
- Adopter
- Shelter
- Main (entry point)

## Setup
Compile and run `Main.java` in any Java IDE.
